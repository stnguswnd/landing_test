# 디자인 가이드 — 그린 계열 영어 학원 랜딩 페이지

## 1. 디자인 방향
이 페이지는 기존 영어 학원 랜딩의 정보 전달 구조를 참고하되, 전반적인 분위기는 더 부드럽고 신뢰감 있는 **그린 계열 프리미엄 학습 브랜드**로 재해석한다.

핵심 인상은 다음 4가지를 목표로 한다.
- **신뢰감**: 학부모가 안심할 수 있어야 한다.
- **명확함**: 복잡한 커리큘럼도 쉽게 읽혀야 한다.
- **성장감**: 로드맵과 단계적 향상을 시각적으로 느낄 수 있어야 한다.
- **정제감**: 학원 랜딩이지만 과장 광고처럼 보이지 않아야 한다.

---

## 2. 무드 앤 톤

### 핵심 키워드
- Fresh
- Academic
- Trustworthy
- Structured
- Premium but warm

### 피해야 할 인상
- 지나치게 키즈 카페 같은 느낌
- 장난감 같은 과한 컬러 사용
- 할인 광고형 전단지 느낌
- 정보가 너무 빽빽한 표 디자인

---

## 3. 컬러 시스템

### 3.1 메인 컬러
- **Primary Green**: 브랜드 핵심색
- **Deep Green**: 제목, 강조, 신뢰 영역
- **Mint Green**: 배경 포인트, 부드러운 보조 영역
- **Lime Accent**: 작은 포인트, 상태 강조

### 3.2 추천 팔레트
```txt
Primary Green: #15803D
Deep Green:    #14532D
Mint Surface:  #DCFCE7
Soft Mint:     #F0FDF4
Lime Accent:   #84CC16
Warm Gray:     #F5F5F4
Text Dark:     #1C1917
Text Muted:    #57534E
Border Soft:   #E7E5E4
White:         #FFFFFF
```

### 3.3 사용 원칙
- 메인 CTA는 Primary Green 사용
- 중요한 제목/지표/강조 영역은 Deep Green 사용
- 카드 배경은 White 또는 Soft Mint 사용
- 보조 하이라이트는 Lime Accent를 소량 사용
- 한 화면에서 강한 포인트 컬러는 1~2개만 사용

---

## 4. 타이포그래피

### 4.1 방향
- 국문 가독성 우선
- 제목은 단단하고 또렷하게
- 본문은 너무 얇지 않게

### 4.2 추천 체계
- **Heading**: Pretendard / SUIT / Noto Sans KR 계열
- **Body**: Pretendard 또는 Noto Sans KR
- **English Accent**: Inter 또는 Manrope 정도를 소량 사용 가능

### 4.3 타입 스케일 예시
- Hero H1: 40~56px / 700~800
- Section H2: 28~36px / 700
- Card Title: 18~22px / 600~700
- Body: 15~18px / 400~500
- Small label: 12~14px / 500~600

### 4.4 문장 스타일
- 헤드라인은 짧고 강하게
- 본문은 2~3줄 단위로 끊어 가독성 확보
- 한 문단이 너무 길어지지 않게 유지

---

## 5. 레이아웃 시스템

### 5.1 전체 폭
- Content max width: 1200~1280px 권장
- 본문은 1120px 안쪽에서 안정적으로 구성

### 5.2 섹션 간격
- 데스크탑: 96~140px
- 모바일: 56~80px

### 5.3 그리드
- 데스크탑: 12-column 기반 사고
- 실제 구현은 2열/3열 카드 중심
- 모바일: 1열 스택

### 5.4 여백 원칙
- 카드 내부 패딩은 넉넉하게
- 텍스트 블록 간 위아래 간격을 충분히 둠
- 로드맵/프로세스 영역은 답답하지 않게 행간 확보

---

## 6. 컴포넌트 스타일 가이드

## 6.1 Hero
### 구성
- 왼쪽: 카피 + CTA
- 오른쪽: 브랜드 비주얼 또는 학습 성장 다이어그램
- 배경: 화이트 기반 + 그린 그라디언트 포인트

### 느낌
- 첫 화면부터 복잡한 표를 넣지 말 것
- 신뢰감 있는 한 문장 + 서브 메시지 + 2개 버튼 중심

---

## 6.2 카드
### 카드 스타일
- 배경: 흰색 또는 매우 연한 민트
- 보더: 아주 약하게
- 둥근 모서리: 20~28px
- 그림자: 약하고 부드럽게

### 카드 유형
- 가치 제안 카드
- 프로그램 소개 카드
- 시스템 단계 카드
- FAQ 아코디언 카드

---

## 6.3 배지 / 라벨
- 작은 라운드 pill 형태
- 예: `문해력`, `사고력`, `서술형`, `입학 상담`
- Deep Green 텍스트 + Mint 배경 조합 권장

---

## 6.4 버튼
### Primary Button
- 배경: Primary Green
- 텍스트: White
- hover: 살짝 진해짐
- radius: 9999px 또는 18px

### Secondary Button
- 배경: White
- 보더: Primary Green
- 텍스트: Primary Green

### 버튼 카피 예시
- 상담 신청하기
- 레벨 테스트 예약
- 커리큘럼 보기
- 문의하기

---

## 6.5 섹션 네비게이션
- 상단 메뉴 대신 간단한 인페이지 탭 또는 우측 플로팅 도트 사용
- active 상태만 분명하게 표시
- 화면을 방해하지 않도록 얇고 단정하게 유지

---

## 7. 섹션별 비주얼 가이드

## 7.1 Overview
- 카드 3개 또는 비교 블록 2개
- 아이콘 + 핵심 문장 + 짧은 설명 구조
- 너무 많은 문장 금지

## 7.2 Brand
- 철학을 보여주는 3개 가치 카드
- 필요 시 원형 아이콘 또는 심볼 사용
- “왜 이 학원인가”를 직관적으로 전달

## 7.3 Integrated Roadmap
- 이 섹션은 가장 복잡하므로 시각적 계층 구조가 중요
- 색상은 단계별로 진해지거나 옅어지는 방식 사용
- 표를 그대로 복붙하는 느낌보다 **재설계된 카드형 로드맵**을 우선
- 모바일에서는 탭/아코디언으로 쪼개기

## 7.4 Learning System
- step flow 다이어그램 추천
- `수업 전 → 수업 중 → 수업 후 → 평가 → 피드백` 흐름이 보이게
- 화면 모형(mockup) 또는 디바이스 프레임을 소량 사용 가능

## 7.5 Admission Process
- 4~5단계 타임라인
- 각 단계는 아이콘 + 제목 + 설명 1줄
- CTA와 자연스럽게 연결

---

## 8. 이미지 사용 가이드

### 권장 이미지 유형
- 실제 수업 분위기 사진
- 학습 몰입 장면
- 교사와 학생 상호작용 장면
- 디지털 학습/피드백을 암시하는 화면 구성

### 이미지 스타일
- 채도 과하지 않게 보정
- 그린 톤과 조화되게 색감 맞춤
- 혼잡한 배경 이미지는 피함

### 사용 주의
- 한 섹션에 이미지가 너무 많지 않게
- 정보 전달용 표/도식과 실제 사진이 충돌하지 않게

---

## 9. 아이콘 가이드
- 아웃라인 또는 단순 면 아이콘 사용
- 스타일 혼합 금지
- 한 세트로 통일
- 너무 귀여운 캐릭터형 아이콘은 지양

---

## 10. 일러스트/도형 가이드
- 원, 곡선, 라운드 사각형 기반으로 부드러운 느낌 형성
- 배경 장식은 저채도/저대비로 사용
- 의미 없는 도형 남발 금지

---

## 11. 모션 가이드
- 등장 애니메이션은 부드럽고 짧게
- 카드 hover는 미세한 lift 정도만
- 섹션 전환 시 과도한 parallax 금지
- 교육 브랜드이므로 안정감이 우선

---

## 12. 반응형 디자인 가이드

### 모바일
- 텍스트 먼저, 시각 요소는 보조
- 1열 스택 우선
- 로드맵은 카드/아코디언으로 재구성
- CTA는 하단 고정 버튼 또는 중간중간 반복 배치

### 태블릿
- 2열 카드 구조 적극 활용
- 너무 넓은 여백보다 정보 균형 유지

### 데스크탑
- 섹션마다 중심선이 느껴지는 정돈된 구조
- 좌우 균형을 맞추되, 한쪽으로 치우친 장식은 자제

---

## 13. 카피 배치 원칙
- 제목 → 한 줄 설명 → 시각 요소 → CTA 순서 유지
- 한 화면 안에서 메시지가 2개 이상 경쟁하지 않게
- 각 섹션은 “이 섹션에서 사용자가 딱 하나만 기억할 것”을 기준으로 설계

---

## 14. 신뢰 요소 표현 가이드
다음 요소는 과하지 않게 배치한다.
- 단계별 학습 로드맵
- 체계적인 관리 프로세스
- 상담/테스트 절차 명확성
- 디지털 학습 연계
- FAQ를 통한 불안 해소

숫자나 성과를 넣을 경우, 근거 없는 과장 표현은 피한다.
